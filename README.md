### ⚡ How It Works — 15 Steps, 380ms, Zero Compromise

```mermaid
sequenceDiagram
    actor S as 🎖️ Sender (BSF-001)
    participant EE as 🔐 Encryption Engine
    participant MN as 🕸️ Mesh Network
    participant R1 as 📡 Relay Node 1
    participant R2 as 📡 Relay Node 2
    participant DE as 🔓 Decryption Engine
    actor R as 🎖️ Receiver (ARMY-005)

    Note over S,R: ⏱️ Total Time: ~380ms | 🔒 Zero Plaintext Ever Transmitted

    S->>EE: 1. Compose plaintext message
    EE->>EE: 2. Generate random AES-256 session key
    EE->>EE: 3. Encrypt message with AES-256-CBC (7ms)
    EE->>EE: 4. Encrypt AES key with receiver's RSA-2048 public key
    EE->>EE: 5. Sign packet with sender's RSA private key (SHA-256)
    EE->>MN: 6. Submit encrypted + signed packet

    Note over MN: 🧠 Intelligent Routing Begins

    MN->>MN: 7. BFS pathfinding across mesh topology (27ms)
    MN->>MN: 8. Filter nodes by battery level (>30% threshold)
    MN->>MN: 9. Calculate altitude-adjusted signal strength
    MN->>R1: 10. Forward encrypted packet (hop 1)
    R1->>R1: 11. Verify relay integrity, update TTL
    R1->>R2: 12. Forward encrypted packet (hop 2)
    R2->>DE: 13. Deliver to destination node

    Note over DE: 🔓 Decryption & Verification

    DE->>DE: 14. Verify RSA digital signature (6ms)
    DE->>DE: 15. Decrypt AES session key with private key
    DE->>DE: 16. Decrypt ciphertext → plaintext
    DE->>R: 17. ✅ Deliver verified, authentic message
```

**Routing Score Formula:**
```
PathScore = (MinBattery × 0.4) + (InverseHops × 0.3) + (SignalStrength × 0.3)

Selected Path: NODE-1 → NODE-4 → NODE-3
  PathScore = (88 × 0.4) + (100/2 × 0.3) + (95 × 0.3) = 35.2 + 15 + 28.5 = 78.7 ✅

Avoided Path: NODE-1 → NODE-2 → NODE-3
```
